# 📊 Dữ liệu Cuộc thi (Dataset)  

Do quy định về kích thước tệp tin của GitHub, bộ dữ liệu gốc không được tải trực tiếp lên repository này.  

### 🔗 Hướng dẫn tải dữ liệu:  
1. Truy cập vào đường dẫn sau: [Tải Dataset tại đây](https://www.kaggle.com/competitions/datathon-2026-round-1/data)  
2. Tải toàn bộ 15 file `.csv` về máy.  
3. Giải nén và đặt tất cả các file vào thư mục `/data/raw/` (phần hướng dẫn trong notebook/README.md) để mã nguồn có thể nhận diện đúng đường dẫn.  

### 📂 Danh sách các bảng dữ liệu chính cần có:  
* **Nhóm Master:** `products.csv`, `customers.csv`, `promotions.csv`, `geography.csv`.  
* **Nhóm Transaction:** `orders.csv`, `order_items.csv`, `payments.csv`, `shipments.csv`, `returns.csv`, `reviews.csv`.  
* **Nhóm Analytical:** `sales.csv` (Dữ liệu train từ 04/07/2012 - 31/12/2022).  
* **Nhóm Operational:** `inventory.csv`, `web_traffic.csv`.
